Utilize a Tag header

Utilizando o Header

O header é utilizado para representar o cabeçalho de um documento ou seção declarado no HTML. Nele podemos inserir elementos da tag h1 a h6, até elmentos para representar imagens, parágrafos ou mesmo listas de navegação.

Utilize a Tag article

Usando o Article

Utilizamos o elemento article quando precisamos declarar um conteúdo que não precisa de outro para fazer sentido em um documnento HTML, por exemplo, um artigo em um blog. É recomendado identificar cada article com um título.

Utilize a Tag section

Utilizando o Section

Seção 1

O elemento section representa uma seção dentro de um documento e geralmente contém um título, o qual é definifo por meio de um dos elementos entre h1 e h6, Podemos utilizar o section, por exemplo, para descrever as seções/tópicos de um documento.

Utilizando o Div

A tag div é uma das tags mais utilizadas no HTML e é essencial para criar layouts e estruturas em um site. Ela é utilizada para dividir o conteúdo em seções lógicas e organizá-las de forma visualmente agradável.
